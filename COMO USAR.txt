================================================================
  DASHBOARD SCM SEMANAL  -  COMO GERAR
================================================================

PASSO A PASSO (toda semana) — SEM PRECISAR DE PYTHON:

  1. Baixe "Gerar Dashboard SCM.exe" (aba Releases do repositório) e
     coloque nesta mesma pasta, junto com a planilha da semana.
  2. Coloque a planilha nova "Relatório de SCs ...xlsx" nesta pasta.
  3. Dê DUPLO-CLIQUE em  ->  "Gerar Dashboard SCM.exe"
  4. Abre uma janela: escolha a planilha da semana e clique Abrir.
       (dica: você também pode ARRASTAR o .xlsx para cima do .exe)
  5. Pronto! O dashboard abre sozinho no navegador e fica salvo em:
       WK\Dashboard SCM WK<numero-da-semana>.html

  Não precisa instalar Python, VSCode nem nada — o .exe já tem
  tudo embutido. Só baixar e usar.

----------------------------------------------------------------
ALTERNATIVA (para quem já tem Python instalado / for mexer no código)

  Em vez do .exe, dá pra rodar "Gerar Dashboard.bat", que chama o
  "gerar_dashboard.py" (requer Python + pandas + openpyxl,
  ver requirements.txt). Mesmo passo a passo acima, só troca o
  arquivo do passo 3.

----------------------------------------------------------------
O QUE O DASHBOARD FAZ

  * Todos os números são calculados automaticamente da planilha
    (abas SCM e SC7). Nada é digitado à mão.

  * TUDO É CLICÁVEL: clique num KPI, numa barra (mês, departamento,
    comprador), numa fatia do aging, etc. -> abre uma tabela com
    TODOS os registros por trás daquele número.
      - Buscar, ordenar, "Ver todos os campos" e "Exportar CSV".

  * Botão Individual / Por Time nos compradores:
      Time Miguel = Miguel + Adrya   |   Time Davi = Davi + Luis Gabriel

  * Aba "Comparar Semanas": a cada semana gerada, o sistema guarda
    um resumo e monta sozinho a comparação (tendência + WK atual vs
    anterior). Quanto mais semanas você gerar, mais rica fica.

----------------------------------------------------------------
COMPARTILHAR O DASHBOARD COM OUTRAS PESSOAS

  O arquivo gerado "WK\Dashboard SCM WK##.html" e AUTOSSUFICIENTE:
  todos os dados E os graficos estao embutidos dentro dele.

  Para enviar (Teams, e-mail, WhatsApp, etc.) basta mandar
  ESSE UNICO arquivo .html. A pessoa so precisa dar duplo-clique
  para abrir no navegador - nao precisa de internet, nem de Python,
  nem da pasta "assets".

  IMPORTANTE: dashboards gerados ANTES desta versao podem abrir
  vazios (sem graficos) quando enviados sozinhos. Basta REGERAR
  (rodar o .bat de novo) para obter a versao autossuficiente.

----------------------------------------------------------------
AJUSTES (opcional, para quem mexe no código)

  Abra "gerar_dashboard.py" e edite o bloco CONFIG no topo:
    - PERIODO_INI / PERIODO_FIM ... período de análise
    - SLA_ATRASO_DIAS ........... limite de atraso (padrão 15 dias)
    - COMPRADORES ............... compradores do time e seus apoios
    - WK_OVERRIDE ............... força um número de semana específico

----------------------------------------------------------------
ARQUIVOS DO SISTEMA (não apague)

  Gerar Dashboard SCM.exe ..... versão pronta p/ usar, sem Python (Releases)
  gerar_dashboard.py .......... o programa-fonte que lê a planilha e gera tudo
  template_dashboard.html ..... o "molde" visual do dashboard
  Gerar Dashboard.bat ......... atalho p/ quem roda via Python
  WK\assets\ .................. bibliotecas dos gráficos (funciona offline)
  WK\data\ ................... histórico das semanas (para o comparativo)
  WK\Dashboard SCM WK##.html .. dashboards gerados (um por semana)

Requisito: nenhum, se usar o .exe. Só quem for mexer no código
precisa de Python instalado (com pandas e openpyxl).
